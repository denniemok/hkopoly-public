# 🏙️ 香江大富翁 - 港版線上大富翁

**HKopoly** 係一個受 [Richup.io](https://richup.io) (大富翁嘅線上替代品) 啟發嘅香港主題網頁版多人棋盤遊戲。  
由深水埗到南區、粉嶺到大嶼山，地名、港鐵線、公用事業都換成大家熟悉嘅香港元素。

**零廣告、零課金、無 VIP 制度。**  
純粹為咗同朋友、網友一齊開局，輕鬆玩返鋪港味大富翁。

## ✨ 特色功能

- **💚 永遠免費：** 
  無廣告、無內購、無隱藏收費，單純為社群分享而設。

- **🎮 即開即玩：** 
  免下載、免註冊，開網頁就可以入房開局。

- **📱 手機友善：** 
  響應式介面適配手機同平板，隨時隨地開瀏覽器即可開局。

- **🏘️ 香港地圖：** 
  目前提供「市區」及「新界及離島」兩款本地棋盤。

- **📜 規則完整：** 
  支援雙骰、坐監、機會、社區基金、起樓、抵押、拍賣等經典規則。

- **⚡ 多人對戰：** 
  支援 2 至 10 人同場，棋盤、聊天室、戰況紀錄即時同步。

- **🤝 自由交易：** 
  玩家可提出混合交易，例如「地皮＋現金＋出獄許可證」。如放棄買地，會即時觸發限時拍賣。

- **💡 交易輔助：** 
  與電腦或託管玩家交易時預設顯示物業估值提示，方便議價。房主可於大廳關閉以提升難度。

- **🤖 精明電腦玩家：** 
  大廳可加入電腦補位。佢哋會喺拍賣中策略性出價、靈活談判交易，欠款時亦會賣樓抵押或提出清倉交易還債。

- **🤖 電腦託管模式：** 
  由電腦代你操作。玩家離線或行開時會自動啟用，你亦可隨時手動開啟或關閉，避免長局因掛機而爛尾。

- **👀 觀戰與重連：** 
  朋友可隨時入場觀戰，斷線後亦可快速重新加入遊戲。

- **⚙️ 自訂家規：** 
  房主可於大廳調整地圖、人數上限（2 至 10 人）、起始資金（HK$500 至 HK$3,000）、公開或私人房間、觀戰權限、玩家洗牌，以及閒置多久後自動託管（1 至 10 分鐘）等設定。

- **🔊 遊戲音效：** 
  擲骰、交易、拍賣等情境均有音效，並可調整音量；設定會儲存於瀏覽器。

- **🌐 雙語介面：** 
  提供英文及繁體中文（香港）完整介面，分別於 hkopoly.com 及 zh.hkopoly.com。

## 🧩 快速開始

> [!TIP]
> 如果房間內所有真人玩家都離開，伺服器會喺 5 分鐘後自動關閉房間。英文版同中文版係兩個唔同嘅伺服器，房間同玩家唔互通。

無需安裝，三步開局：

1. **加入或建立遊戲**  
   選擇「加入遊戲」或「建立遊戲」。

2. **輸入玩家名稱**  
   用你想顯示嘅名稱入場。

3. **開始遊戲**  
   房主選好地圖後即可開波。輪到你時：  
   **擲骰 → 買地 / 交租 / 交易 → 完結回合**

最後仍未破產嘅玩家就係贏家！

## 🚀 未來計劃

HKopoly 會持續更新，以下係暫定方向：

  加入更多本地化主題棋盤。歡迎社群透過 Pull Request 提交新地圖。

  讓房主調整坐監回合、租金倍率等進階家規（地圖、人數、起始資金、觀戰等已可於大廳設定）。

  改善 UI 佈局，加入更流暢嘅擲骰、買地及交易動畫。

- **📈 本地戰績紀錄：**
